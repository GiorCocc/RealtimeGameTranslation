# Realtime Game Translation (Upcoming Full Version)

Realtime Game Translation is a desktop app that helps you understand games in other languages while you play.

It reads on-screen text (like dialogue, menus, and UI elements), translates it in real time, and shows the translated text in a clean overlay on top of the game window.  
The goal is to make story-heavy games, RPGs, and foreign releases more accessible without breaking immersion.

## What the full version is designed for

- Playing games that don’t support your language
- Following dialogues and quest text more comfortably
- Keeping the original game visible while seeing live translations
- Fast, lightweight translation flow focused on readability

## Planned user experience

- One-click start
- Live translation overlay during gameplay
- Clear and minimal visual style
- Easy language selection and quick toggles
- Performance-focused behavior for long play sessions
